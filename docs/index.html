Lab 1: The Artemis board and Bluetooth.
In the first experiment, I set up a basic environment for subsequent experiments and used examples to test the environment. In addition, I also used BLE to establish Bluetooth communication between the Artemis board and the computer, and wrote multiple examples to test the transmission and reception of data in different ways.

Lab One Information

To set up my computer for Python development with the Artemis board: 1. **Ensure Python 3.9+ and pip 21.0+** are installed. Check versions with `python3 --version` and `pip --version`. 2. **Install Python and pip** using your system's package manager for Linux/FreeBSD users. 3. **Create a virtual environment** named "FastRobots_ble" in your project directory using `python3 -m venv FastRobots_ble`. 4. **Activate the virtual environment** with `.\FastRobots_ble\Scripts\activate` (Windows) or its Linux/macOS equivalent. 5. **Install necessary Python packages** (`numpy`, `pyyaml`, `colorama`, `nest_asyncio`, `bleak`, `jupyterlab`) using pip within the virtual environment. 6. **Set up the Lab 2 codebase** by downloading, unzipping, and ensuring the `ble_python` directory is in your project directory. 7. **Start the Jupyter server** with `jupyter lab` to work on Python code in Jupyter notebooks. This streamlined process prepares your computer for developing Bluetooth communication projects with the Artemis board using Python.

Artemis Board Setup Summary

  1. Obtain Artemis MAC Address: Get the BLE MAC address from the Artemis board documentation or by running a sketch.
  2. Update connections.yaml: Replace artemis_address with the obtained MAC address.
  3. Generate UUID: Use a Python script with from uuid import uuid4; print(uuid4()) to generate a UUID.
  4. Update Files with UUID:
    • For ble_arduino.ino: Replace the BLEService UUID.
    • For connections.yaml: Update ble_service with the new UUID.
  5. Save Changes: Ensure all files are saved with updates.
  6. Test BLE Connection: Re-test to confirm everything is set up correctly.

Note: Adjust base_ble.py for platform-specific configurations if necessary.
